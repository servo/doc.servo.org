    byte_order: ByteOrder,

) -> Result<()>

Attempts to read a number of the represented values from the given stream with the given byte order.

n values will be are stored in target, or an error will be returned. target vector may be modified even if this method returns an error.

fn read_from_u32(
    source: [u8; 4],
    byte_order: ByteOrder,
    n: usize,
    count: usize
) -> Option<Self::Repr>

Reads the nth represented value inside source.

If the value can be read successfully (n < count, the represented type is smaller than or equal to u32, etc.), returns Some(value), otherwise returns None.
