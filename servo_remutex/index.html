An implementation of re-entrant mutexes.

Re-entrant mutexes are like mutexes, but where it is expected that a single thread may own a lock more than once. It provides the same interface as https://github.com/rust-lang/rust/blob/5edaa7eefd76d4996dcf85dfc1c1a3f737087257/src/libstd/sys_common/remutex.rs so if those types are ever exported, we should be able to replace this implemtation.
