Configuration-independent validation of a ClientHello message.

This represents the first part of the ClientHello handling, where we do all validation that doesn’t depend on a ServerConfig being available and extract everything needed to build a ClientHello value for a [ResolvesServerConfig]/ResolvesServerCert].

Note that this will modify data.sni even if config or certificate resolution fail.
