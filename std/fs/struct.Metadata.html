pub struct Metadata(_);

Metadata information about a file.

This structure is returned from the metadata function or method and represents known metadata about a file such as its permissions, size, modification times, etc.

Methods

impl Metadata

fn is_dir(&self) -> bool

Returns whether this metadata is for a directory.

fn is_file(&self) -> bool

Returns whether this metadata is for a regular file.

fn len(&self) -> u64

Returns the size of the file, in bytes, this metadata is for.

fn permissions(&self) -> Permissions

Returns the permissions of the file this metadata is for.

fn accessed(&self) -> u64

Returns the most recent access time for a file.

The return value is in milliseconds since the epoch.

fn modified(&self) -> u64

Returns the most recent modification time for a file.

The return value is in milliseconds since the epoch.
