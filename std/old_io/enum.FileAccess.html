pub enum FileAccess {
    Read,
    Write,
    ReadWrite,
}

Access permissions with which the file should be opened. Files opened with Read will return an error if written to.

Variants

Read

Read-only access, requests to write will result in an error

Write

Write-only access, requests to read will result in an error

ReadWrite

Read-write access, no requests are denied by default
