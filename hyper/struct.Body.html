Struct hyper::Body[−][src]

pub struct Body {
    kind: Kind,
    extra: Option<Box<Extra>>,
}
Expand description

A stream of Chunks, used when receiving bodies.

A good default Payload to use in many applications.

Also implements futures::Stream, so stream combinators may be used.

Fields

kind: Kindextra: Option<Box<Extra>>

Keep the extra bits in an Option<Box<Extra>>, so that Body stays small in the common case (no extras needed).
