Adds the display items for this fragment to the given display list.

Arguments:

  • display_list: The display list to add display items to.
  • layout_context: The layout context.
  • dirty: The dirty rectangle in the coordinate system of the owning flow.
  • stacking_relative_flow_origin: Position of the origin of the owning flow with respect to its nearest ancestor stacking context.
  • relative_containing_block_size: The size of the containing block that position: relative makes use of.
  • clip: The region to clip the display items to.
