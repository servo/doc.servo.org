pub fn pref_map() -> &'static Preferences<'static, Prefs>
Expand description

Access preferences using their String keys. Note that the key may be different from the static path because legacy keys contain hyphens, or because a preference name has been renamed.

When retrieving a preference, the value will always be a PrefValue. When setting a value, it may be a PrefValue or the type that converts into the correct underlying value; one of bool, i64, f64 or String.
