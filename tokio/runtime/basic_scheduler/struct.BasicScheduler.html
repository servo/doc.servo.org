}
Expand description

Executes tasks on the current thread

Fields

inner: Mutex<Option<Inner<P>>>

Inner state guarded by a mutex that is shared between all block_on calls.

notify: Notify

Notifier for waking up other threads to steal the parker.

spawner: Spawner

Sendable task spawner

context_guard: Option<EnterGuard>

This is usually None, but right before dropping the BasicScheduler, it is changed to Some with the context being the runtime’s own context. This ensures that any tasks dropped in the BasicSchedulers destructor run in that runtime’s context.
