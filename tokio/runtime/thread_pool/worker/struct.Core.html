Expand description

Core data

Fields

tick: u8

Used to schedule bookkeeping tasks every so often.

lifo_slot: Option<Notified<Arc<Shared>>>

When a task is scheduled from a worker, it is stored in this slot. The worker will check this slot for a task before checking the run queue. This effectively results in the last scheduled task to be run next (LIFO). This is an optimization for message passing patterns and helps to reduce latency.

run_queue: Local<Arc<Shared>>

The worker-local run queue.

is_searching: bool

True if the worker is currently searching for more work. Searching involves attempting to steal from other workers.

is_shutdown: bool

True if the scheduler is being shutdown

park: Option<Parker>

Parker

Stored in an Option as the parker is added / removed to make the borrow checker happy.

stats: WorkerStatsBatcher

Batching stats so they can be submitted to RuntimeStats.

rand: FastRand

Fast random number generator.
