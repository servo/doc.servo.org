Expand description

State shared across all workers

Fields

remotes: Box<[Remote]>

Per-worker remote state. All other workers have access to this and is how they communicate between each other.

inject: Inject<Arc<Shared>>

Submits work to the scheduler while not currently on a worker thread.

idle: Idle

Coordinates idle workers

owned: OwnedTasks<Arc<Shared>>

Collection of all active tasks spawned onto this executor.

shutdown_cores: Mutex<Vec<Box<Core>>>

Cores that have observed the shutdown signal

The core is not placed back in the worker to avoid it from being stolen by a thread that was spawned as part of block_in_place.

before_park: Option<Arc<dyn Fn() + Send + Sync>>

Callback for a worker parking itself

after_unpark: Option<Arc<dyn Fn() + Send + Sync>>

Callback for a worker unparking itself

stats: RuntimeStats

Collects stats from the runtime.
