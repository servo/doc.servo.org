enum CountLatchKind {
    Stealing {
        latch: CoreLatch,
        registry: Arc<Registry>,
        worker_index: usize,
    },
    Blocking {
        latch: LockLatch,
    },
}

Variants§

§

Stealing

A latch for scopes created on a rayon thread which will participate in work- stealing while it waits for completion. This thread is not necessarily part of the same registry as the scope itself!

Fields

§latch: CoreLatch
§registry: Arc<Registry>

If a worker thread in registry A calls in_place_scope on a ThreadPool with registry B, when a job completes in a thread of registry B, we may need to call notify_worker_latch_is_set() to wake the thread in registry A. That means we need a reference to registry A (since at that point we will only have a reference to registry B), so we stash it here.

§worker_index: usize

The index of the worker to wake in registry

§

Blocking

A latch for scopes created on a non-rayon thread which will block to wait.

Fields

§latch: LockLatch
