    HeaderTableSize(u32),
    EnablePush(u32),
    MaxConcurrentStreams(u32),

    MaxHeaderListSize(u32),
    EnableConnectProtocol(u32),
}
Expand description

An enum that lists all valid settings that can be sent in a SETTINGS frame.

Each setting has a value that is a 32 bit unsigned integer (6.5.1.).

source

fn load(raw: &[u8]) -> Option<Setting>

Creates a new Setting by parsing the given buffer of 6 bytes, which contains the raw byte representation of the setting, according to the “SETTINGS format” defined in section 6.5.1.

The raw parameter should have length at least 6 bytes, since the length of the raw setting is exactly 6 bytes.

Panics

If given a buffer shorter than 6 bytes, the function will panic.
