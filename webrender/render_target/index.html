IDEAL_MAX_TEXTURE_DIMENSION

According to apitrace, textures larger than 2048 break fast clear optimizations on some intel drivers. We sometimes need to go larger, but we try to avoid it. This can go away when proper tiling support lands, since we can then split large primitives across multiple textures.
