fn next(&mut self) -> XmlEvent

Pulls and returns next XML event from the stream.

If returned event is xml::event::Error or xml::event::EndDocument, then further calls to this method will return this event again.

fn events<'a>(&'a mut self) -> Events<'a, B>

Returns an iterator over XML events.

When the next event is xml::event::Error or xml::event::EndDocument, then it will be returned by the iterator once, and then it will stop producing events.
