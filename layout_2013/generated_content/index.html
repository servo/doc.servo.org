Module layout_2013::generated_content

Expand description

The generated content assignment phase.

This phase handles CSS counters, quotes, and ordered lists per CSS § 12.3-12.5. It cannot be done in parallel and is therefore a sequential pass that runs on as little of the flow tree as possible.
