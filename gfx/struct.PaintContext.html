The rectangle that this context encompasses in page coordinates.

screen_rect

The rectangle that this context encompasses in screen coordinates (pixels).

clip_rect

The clipping rect for the stacking context as a whole.

transient_clip

The current transient clipping region, if any. A "transient clipping region" is the clipping region used by the last display item. We cache the last value so that we avoid pushing and popping clipping regions unnecessarily.
