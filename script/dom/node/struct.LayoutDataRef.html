unsafe fn borrow_unchecked(&self) -> *const Option<LayoutData>

Borrows the layout data immutably, asserting that there are no mutators. Bad things will happen if you try to mutate the layout data while this is held. This is the only thread- safe layout data accessor.

fn borrow<'a>(&'a self) -> Ref<'a, Option<LayoutData>>

fn borrow_mut<'a>(&'a self) -> RefMut<'a, Option<LayoutData>>

Borrows the layout data mutably. This function is not thread-safe.
