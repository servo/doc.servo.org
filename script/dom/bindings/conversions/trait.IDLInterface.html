pub trait IDLInterface: MarkerTrait {
    fn get_prototype_id() -> ID;
    fn get_prototype_depth() -> usize;
}

A trait to retrieve the constants necessary to check if a JSObject implements a given interface.

Required Methods

fn get_prototype_id() -> ID

Returns the prototype ID.

fn get_prototype_depth() -> usize

Returns the prototype depth, i.e., the number of interfaces this interface inherits from.
