pub(crate) struct Task {
    state: AtomicUsize,
    blocking: AtomicUsize,
    next_blocking: AtomicPtr<Task>,
    pub reg_worker: Cell<Option<u32>>,
    pub reg_index: Cell<usize>,
    future: UnsafeCell<Option<Spawn<Box<dyn Future<Item = (), Error = ()> + Send + 'static>>>>,
}
Expand description

Harness around a future.

This also behaves as a node in the inbound work queue and the blocking queue.

Fields

state: AtomicUsize

Task lifecycle state

blocking: AtomicUsize

Task blocking related state

next_blocking: AtomicPtr<Task>

Next pointer in the queue of tasks pending blocking capacity.

reg_worker: Cell<Option<u32>>

ID of the worker that polled this task first.

This field can be a Cell because it’s only accessed by the worker thread that is executing the task.

The worker ID is represented by a u32 rather than usize in order to save some space on 64-bit platforms.

reg_index: Cell<usize>

The key associated with this task in the Slab it was registered in.

This field can be a Cell because it’s only accessed by the worker thread that has registered the task.

future: UnsafeCell<Option<Spawn<Box<dyn Future<Item = (), Error = ()> + Send + 'static>>>>

Store the future at the head of the struct

The future is dropped immediately when it transitions to Complete
