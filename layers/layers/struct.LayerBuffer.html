Fields

native_surface

The native surface which can be shared between threads or processes. On Mac this is an IOSurface; on Linux this is an X Pixmap; on Android this is an EGLImageKHR.

rect

The rect in the containing RenderLayer that this represents.

screen_pos

The rect in pixels that will be drawn to the screen.

resolution

The scale at which this tile is rendered

stride

NB: stride is in pixels, like OpenGL GL_UNPACK_ROW_LENGTH.

painted_with_cpu

Whether or not this buffer was painted with the CPU rasterization.

content_age

The content age of that this buffer request corresponds to.
