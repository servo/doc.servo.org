Module deflate::zlib[−][src]

Expand description

This module contains functionality for generating a zlib header.

The Zlib header contains some metadata (a window size and a compression level), and optionally a block of data serving as an extra dictionary for the compressor/decompressor. The dictionary is not implemented in this library. The data in the header aside from the dictionary doesn’t actually have any effect on the decompressed data, it only offers some hints for the decompressor on how the data was compressed.
