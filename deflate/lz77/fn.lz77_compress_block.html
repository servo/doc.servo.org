Expand description

Compress a slice with lz77 compression.

This function processes one window at a time, and returns when there is no input left, or it determines it’s time to end a block.

Returns the number of bytes of the input that were consumed, a status describing whether there is no input, it’s time to finish, or it’s time to end the block, and the position of the first byte in the input buffer that has not been output (but may have been checked for matches).
