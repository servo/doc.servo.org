Enum style::properties::PropertyId

source · 
pub enum PropertyId {
    NonCustom(NonCustomPropertyId),
    Custom(Name),
}
Expand description

Representation of a CSS property, that is, either a longhand, a shorthand, or a custom property.

Variants§

§

NonCustom(NonCustomPropertyId)

An alias for a shorthand property.

§

Custom(Name)

A custom property.
