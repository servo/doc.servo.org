#[repr(C, u8)]
pub enum GenericLengthOrNumber<L, N> {
    Number(N),
    Length(L),
}
Expand description

A generic <length> | <number> value for the -moz-tab-size property.

Variants

Number

A number.

NOTE: Numbers need to be before lengths, in order to parse them first, since 0 should be a number, not the 0px length.
