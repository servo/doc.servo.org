fn build_frame(&self, sample: Sample) -> Result<VideoFrame, ()>

Returns the Player’s Frame to be consumed by the API user.

The implementator of this method will map the sample’s buffer to the rendering appropiate structure. In the case of OpenGL-based renders, the Frame, instead of the raw data, will transfer the texture ID.

Arguments
  • sample - the GStreamer sample with the buffer to map
